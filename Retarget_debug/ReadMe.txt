%% README.TXT

1. run Retargeting.bat
2. type any images name (including directory) that you want to retarget. (ex: ./imgs/space.jpg)
3. press 'o' button if you want to save current retargeted image to 'bmp' file format
4. retargeted image will be saved as 'Retargeted.bmp'
